# Raw Data Repository

Due to file size constraints on GitHub, the raw RNA-seq matrices, barcode, and feature files (approximately 700 MB) utilized in this pipeline are hosted on Zenodo. 

**Access the aggregated dataset here:**
[DOI: 10.5281/zenodo.19570951](https://doi.org/10.5281/zenodo.19570951)

This Zenodo repository serves as a permanent, aggregated backup to guarantee continuous reproducibility in the event of upstream database connectivity issues. 

To utilize this local backup, download the archive from Zenodo and extract the individual `_data` directories directly into the root of this project before building and executing the Docker container.
